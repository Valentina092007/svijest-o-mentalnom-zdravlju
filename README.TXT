Ova web-stranica je napravljena kao projekt za aktivnost Code Week: Coding for social changes.
Namjena web-stranice je potiči druge o važnosti brige za mentalno zdravlje i njegovo održavanje.
Web-stranica nudi razne kontakte za savjetovanje u vezi bilo kakvih mentalnih poteškoća i centre za pomoć
u slučaju kriznih situacija.
Također sadrži i nekoliko korisnih savjeta za održavanje mentalnog zdravlja.
